#+title: simulacrum.el
Inject arbitrary forms into the event stream.

* Installation
#+begin_src elisp
  (use-package simulacrum
    :vc (:url "https://github.com/ErikPrantare/simulacrum.el"
              :branch "master"
              :rev :newest))
#+end_src

Forms are injected by calling ~simulacrum-evaluate~ with the lisp form
that should be injected into the event stream.  The form will be
evaluated as part of the usual command loop.
